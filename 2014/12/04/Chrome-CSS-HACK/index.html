Chrome Css Hack 笔记

  • Chrome

本文转载自: Chrome Css Hack 笔记

1. Chrome浏览器最小中文字体的设置

似乎谷歌浏览器会限制中文字体的大小，最小值为12px。下面的代码可以解决这一问题。

.small-text {   font-size: 8px   -webkit-text-size-adjust:none; }

2. HTML5表单元素placeholder属性的文本的样式设置

HTML5为表单的input、textarea元素提供了placeholder属性，有时候我们需要设置占位文本的样式，目前似乎需要一些兼容技巧。

::-webkit-input-placeholder,    input:-moz-placeholder {    color: #c4c0c0;  }

3. 去除Chrome浏览器input、textarea等元素的蓝色边框

input, textarea {   outline: none; }

4. webkit内核浏览器<input type="search">元素样式

webkit内核的浏览器为type=”search”的input元素提供了特殊的样式，通过下面的css代码可以变回原本的input元素的样式。

input[type="search"] {   -webkit-appearance: textfield; }

更多-webkit-appearance的可选属性详见WebKit-Specific Style: -webkit-appearance。
