HTML5 新特性（六） input

  • HTML5

       在HTML5中，对input标签进行了加强。新增加了一些新的类型。详细请查看:HTML 5 <input> type 属性

1. number

type=number 的HTML表单元素是可以用按键的方式改变文本框中的值。

<input type="number" value="1">

2. range

type=range 意为可以选择某个值的区域范围。

<input type="range" value="50" max="1000" min="0" step="10">

3. date

  • date
<input type="date" id="aa" value="2011-11-14">
  • datetime
<input type="datetime" value="2011-11-14">
  • datetime-local
<input type="datetime-local" value="2011-11-14T21:12">
  • month
<input type="month"  value="2011-11">
  • week
<input type="week" value="2011-W12" />
  • time
<input type="time" value="09:56" />

4. color

type=color意为颜色选择器控件。

<input type="color" value="#34538b">

5. search

type=search意为搜索功能。

<input type="search" >

6. email

<input type="email" >

7. tel

<input type="tel" value="13645621385">

8. url

<input type="url" value="http://www.baidu.com">
