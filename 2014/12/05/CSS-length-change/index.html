CSS 长度转换

  • CSS

        我们一般写CSS定义DOM元素的宽、高等一些需要给定值的属性时。我们一般都是使用px，或者一些使用em。但是，CSS里面的长度单位真的就只有这两种？今天，我们就来扒一扒CSS里面的那些长度单位。

px

像素（Pixel）,相对于设备的长度单位，像素是相对于显示器屏幕分辨率而言的。

em

相对长度单位。相对于当前对象内文本的字体尺寸。如当前行内文本的字体尺寸未被人为设置，则相对于浏览器的默认字体尺寸。即％，是相对单位，是一个相对长度单位，最初是指字母M的宽度，故名em。现指的是字符宽度的倍数，用法类似百分比，如：0.8em, 1.2em,2em等。通常1em=16px。，一般用来测量长度的通用单位(例如元素周转的页边空白和填充),当用于指定字体大小时,em单位是指父元素的字体大小。在一个页面上给定了一个父元素的字体大小,这样就可以通过调整一个元素来成比例的改变所有元素大小.它可以自由缩放,比如用来制作可伸缩的样式表。

ex

相对长度单位。相对于字符“x”的高度。此高度通常为字体尺寸的一半。如当前对行内文本的字体尺寸未被人为设置，则相对于浏览器的默认字体尺寸。

pt

点（Point），绝对长度单位。

pc

派卡（Pica），绝对长度单位。相当于我国新四号铅字的尺寸。

in

英寸（Inch），绝对长度单位。

mm

毫米（Millimeter），绝对长度单位。

cm

厘米（Centimeter），绝对长度单位。

相对长度应用

        pt、pc、in、mm、cm 这几个都是绝对长度，与px的用法毫无两样。只是把后面的px单位换成对应的其他单位就好了。下面讲一下em的用法。一般浏览器的 font-size 是16px 的。下面列举一下相对像素间的转换.

px to em:Example: 12px / 16px = .75em
px to %:Example: 12px / 16px 100 = 75%
px to pt:Example: 16px 72 (72 points = 1 inch) / 96 (96 pixels per inch in Windows, 72 in Mac) = 12pt
em to px:Example: .75em 16px = 12px
em to %:Example: .75em 100 = 75%
% to px:Example: 75 16px / 100 = 12px
% to em:Example: 75 / 100 = .75em
pt to px:Example: 12pt 96ppi / 72ppi = 16px

<div>
    这里的字体大小与默认字体大小一样。
    <div style="font-size:51.2px">
            这里的字体大小与51.2px相仿。
            <div style="font-size:1em">这里的字体大小与51.2px相仿。</div>
        </div>
</div>

从上面的代码可以看出，当副容器设置font-size时，子元素使用em来设置字体大小.1em就相当于父容器设置的51.2px。整个页面的width最大为20em，如果屏幕分辨率为1920px我们使用em来充当页面的长度单位的话。那我们需要在<body>标签设置font-size样式，设置font-size为 96px (1920px / 20em = 96px). <body style='font-size:96px'>,那body里面的dom元素长度单位 1em = 96px；
